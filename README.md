## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SubhrjiT/web.git
```

### 2. Navigate to project folder

```bash
cd web
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
